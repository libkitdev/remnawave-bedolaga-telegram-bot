# Настройка платежных систем

Этот документ описывает общий порядок настройки платежных провайдеров в проекте.

## Общая последовательность

1. Получите API-ключи и секреты в личном кабинете провайдера.
2. Заполните переменные в `.env`.
3. Настройте публичный HTTPS webhook URL.
4. Перезапустите сервис и проверьте health endpoint.
5. Выполните тестовый платеж и убедитесь в автозачислении.
